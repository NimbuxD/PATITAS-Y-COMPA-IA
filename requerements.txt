Instrucciones para Configurar y Ejecutar el Proyecto

1. Instalar Python:
   - Descarga e instala Python desde [python.org](https://www.python.org/downloads/).
   - Durante la instalación, asegúrate de seleccionar la casilla "Add python.exe to PATH" para que Python se agregue a la variable de entorno `PATH`.

2. Instalar Flask y Flask-RESTful:
   - Abre una consola (símbolo del sistema, PowerShell, o terminal).
   - Ejecuta el siguiente comando para instalar Flask y Flask-RESTful:

     pip install Flask Flask-RESTful

3. Ejecutar la Aplicación:
   - Navega a la carpeta del proyecto en la terminal.
   - Ejecuta la aplicación con el siguiente comando:

     python app.py
